Филтры

FG4 filter

Использование газовой фазы в линии, проверяемый, пластиковый корпус. Фильтрующий элемент из смоляной бумаги. Доступно для газовой трубы с ᴓ внутри 12 мм. Вихревая система оптимизирована для отделения масляных элементов от газа, поступающего в дренируемый поддон. Фильтр обеспечивает размещение датчиков BOSCH или Continental/Siemens.

FG3 filter

Используется для газовой фазы в линии, не подлежит осмотру, пластиковый корпус. Фильтрующий элемент из смоляной бумаги. Доступно для газовых труб с внутренним диаметром 10-12-14-16 мм.

FU1 filter

Фильтр AUS32® для автомобилей с дизельными двигателями. Новый фильтр был разработан в ответ на появление новой технологии по сокращению выбросов с помощью реагента AdBlue® — химически синтезированного водного раствора с 30-процентным содержанием мочевины, преобразующего оксиды азота в воду и безвредный азот. Компактный размер, гибкая установка Соединения согласно стандарту SAE J2044 Уплотнительный материал EPDM

FU4 filter

Фильтр приложений AUS32® Новый фильтр был разработан в ответ на новую развивающуюся технологию снижения выбросов с помощью агента AUS32 — водного раствора химического синтеза с содержанием мочевины 30 процентов, преобразующего оксиды азота в воду и безвредный азот. Компактный размер, гибкая установка Соединения согласно стандарту SAE J2044 Уплотнительный материал EPDM
